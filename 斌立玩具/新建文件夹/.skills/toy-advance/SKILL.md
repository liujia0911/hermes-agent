# toy-advance

玩具垫资计算技能。

## 触发
算垫资/玩具垫资/计算垫款/算账/算尾款

## 公式
总成交额=SUM(qty*amount)
实际成交额=总成交额-SUM(已关闭)
需垫款90%=实际成交额*0.9
尾款10%=实际成交额*0.1
服务费5%=实际成交额*0.05
税点2%=实际成交额*0.02
剩余尾款=尾款10%-服务费5%-税点2%-|平台服务费|-动账出账

平台服务费=结算CSV col40 支出合计
动账出账=动账CSV出账方向合计

## 订单列映射
麦6/麦7(16列):qty[4],amount[5],status[8]
麦8(20列):qty[6],amount[7],status[13]

## 执行
python .skills/toy-advance/scripts/calculate.py 目录名

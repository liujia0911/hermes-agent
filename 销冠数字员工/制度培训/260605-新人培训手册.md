## 二、业务演进史（为什么我们做今天的事）

### 6月之前：全人工交付模式

旧业务链路：
```
企业接洽 → 建号装修 → 脚本策划 → 拍摄剪辑 → 巨量本地推投流 → 客服承接转化
           ←—————— 全部人工交付 ——————→
```

**结果：**
- ✅ 有效果，能帮企业拿到精准客资
- ❌ 交付极重，团队堆人力
- ❌ 企业月费高昂（服务费+广告费）
- ❌ 只服务有预算的大工厂，小商家无法合作
- ❌ 一旦转化不理想，客户试错成本高，易亏损

---

### 6月起：数字员工自动化交付

新业务链路：
```
上门拍摄（唯一人工环节）→ 数字员工接管 →
    策划脚本 → 剪辑视频 → 定时发布 → 投流优化 → GEO官网内容 → 客询承接
    ←—————— 全自动交付 ——————→
```

**变化：**
| | 旧模式 | 新模式 |
|---|---|---|
| 交付方式 | 全人工 | 1步人工 + 全自动 |
| 收费 | 高月服务费 | 免费使用，按token消耗付费 |
| 客户群 | 只服务大厂 | 大中小商家全覆盖 |
| 团队配置 | 需多人 | 2人核心 + AI工具链 |

## 三、为什么这个模式能成立

### 抖音的真相

- **自然流量已死。** 平台算法比你自己更了解你接下来要买什么。
- **广告竞价透明。** 精准客资市场价50元，你出40元？要么花不出去，要么买到垃圾流量。
- **视频播放量是面子，客询转化是里子。** 商家不需要当网红，需要的是精准客户来咨询。

### 获客的本质：让刚好有需求的人，刚好看到你

**最佳成交场景：** 客户有需求的那一刻，你的内容刚好出现在他面前。

怎么实现？三种路径，我们叫——**四条获客通道**：

| 引擎 | 客户怎么找到你 | 特点 |
|------|--------------|------|
| **广告** | 刷抖音时刷到你的广告 | 快，付费买时间 |
| **搜索** | 在抖音搜行业词，你的视频排第一 | 长期免费，占关键词 |
| **推荐** | 在豆包/Kimi/DeepSeek里问"XX行业哪家好"，你的官网被AI引用推荐 | 新赛道，抢占AI入口 |

> 四个源头汇聚成一个结果：客户不管用什么方式找，都能通向你的生意。

## 四、GEO：新战场，新机遇

### 什么是GEO

SEO是在传统搜索引擎（百度/谷歌）做排名。GEO是在**AI搜索引擎**（豆包、Kimi、DeepSeek、元宝、ChatGPT）里做推荐。

当用户问AI"深圳哪家装修公司靠谱"，AI不是随便回答的——它会抓取和引用互联网上的官网内容作为依据。**谁的内容多、谁的内容好，谁就被AI推荐。**

### 怎么做

```
数字员工自动生成行业文章 → 自动同步客户官网 
    → 搜索引擎抓取官网内容 → AI学习并引用 → 用户问AI时被推荐 → 客户上门
```

核心动作：**建官网，填内容。** 不是以前那种"放个首页就不管了"的官网，而是每天自动更新、被搜索引擎持续收录的内容资产。

### 为什么第三方平台不够

- 抖音/小红书/知乎等平台对营销内容管控越来越严，轻则限流，重则封号
- 内容发在别人平台上，数据是别人的；发在官网上，数据是自己的
- 官网内容能被所有搜索引擎抓取，一次投入，全网受益

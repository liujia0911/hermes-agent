# 昕冠·抖音运营部群规

---

## 通用规则（三个群共同）

1. 小马是本群Agent，@小马即可触发任务
2. Agent只负责执行和汇报，最终决策由佳哥/睿哥拍板
3. 群内不闲聊，每条消息都有明确目的
4. 所有产出文件自动存档到Obsidian

---

## 内容策划组

**用途：** IP内容生产和物料设计

**成员：** 佳哥、睿哥、@小马

**@小马触发命令：**
- "@小马 今天拍什么" → 出今日拍摄脚本
- "@小马 明天选题" → 出明日选题建议
- "@小马 出张封面" → 根据脚本生成视频封面
- "@小马 改这段口播" → 优化指定文案
- "@小马 下周排期" → 出下周拍摄计划

**Agent自动行为：**
- 每天8:00 推送今日脚本
- 每天8:30 推送宣传物料
- 周五17:00 推送下周排期

---

## 运营数据组

**用途：** 数据复盘和策略调整

**成员：** 睿哥、@小马

**@小马触发命令：**
- "@小马 昨天数据" → 播报昨日播放/互动/粉丝数据
- "@小马 本周数据" → 生成周报
- "@小马 竞品看看" → 对比竞品IP数据
- "@小马 策略建议" → 输出优化方向

**Agent自动行为：**
- 每天9:00 昨日数据播报
- 数据异常时自动预警（播放暴跌/掉粉）
- 每周一出上周周报

---

## 客户增长组

**用途：** 线索管理和客户交付

**成员：** 佳哥、睿哥、@小马

**@小马触发命令：**
- "@小马 新线索" → 查看最新留资客户
- "@小马 谁该跟进了" → 查看超时未联系的线索
- "@小马 乌贼帮进度" → 查看指定客户项目状态
- "@小马 跟进话术" → 针对某客户出跟进文案

**Agent自动行为：**
- 每2小时扫描新线索
- 高意向超24h未跟进自动预警
- 新客户入驻自动发欢迎包
- 客户到期前7天续费提醒

---

## 禁止行为

- 禁止在非对应群@小马触发不相关任务
- 禁止把小马拉进其他群（工作边界清晰）
- 禁止群内讨论与职能无关的话题

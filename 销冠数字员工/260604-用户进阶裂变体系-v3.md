## 四、每一层怎么玩

### L1：散客 — 试试看

**怎么进：** 注册即散客，100算力起充（100元）。

**能干什么：**
- AI视频剪辑：上传素材，自动出片
- 广告搭建：一键投放到抖音/视频号
- SEO + GEO：7天免费体验

**不能干什么：**
- 广告盯盘（需要会员）
- 推荐别人赚佣金（需要会员）

**适合谁：** 想先试试效果的老板。100块试一下，比请人吃饭还便宜。

**升级路径：** 累计充值满2000元 → 自动升级会员。100算力够用5-7天，用完觉得好再充。


### L2：会员 — 真正用起来

**怎么进：** 累计充值满2000元（约1-2个月用量）。

**比散客多了什么：**
- 广告盯盘功能全开：AI 24小时盯着你的广告，自动调价、自动换素材
- SEO和GEO永久开放：排名优化不间断
- 完整数据看板：每条视频播放量、每个关键词排名、每分钱花在哪
- **开始赚钱：推荐一个新用户，他充算力你拿10%**

**赚钱怎么算：**
- 你推荐老王进来，老王充了2000算力 → 你拿200元
- 老王每个月都消耗2000算力 → 你每个月拿200元
- 推荐10个老王 → 你每个月被动收入2000元
- 老王自己不续费了 → 你的收入停掉，但你已经赚到前面的了

**为什么比散客更吸引人：**
散客是纯花钱。会员是花着花着开始赚钱——推荐几个人，算力等于白用。

**升级路径：** 累计充值满10000元 → 自动升级高级会员。


### L3：高级会员 — 开始认真赚钱

**怎么进：** 累计充值满10000元（约半年重度使用）。

**比会员多了什么：**
- 推荐奖励从10%升级到15%：同样推荐老王充2000，你拿300（多了50%）
- **间推奖励5%：你推荐的人再推荐的人，你也拿**
- 专属客服：1小时内响应
- 新功能优先体验：比别人早一周用新功能
- 数据导出：下载报表，给客户看效果

**赚钱怎么算：**
- 你推荐老王（会员），老王充2000 → 你拿300（15%）
- 老王又推荐了老李，老李充2000 → 老王拿200（10%），你拿100（5%）
- 老王推荐10个老李 → 你每个月被动收入1000元（光间推）
- 你自己再推荐10个老王 → 300×10 = 3000元/月（直推）
- **合计：月被动收入4000元**

**为什么比会员更吸引人：**
会员只能赚自己推的人。高级会员能赚"别人推的人"——滚雪球开始了。

**升级路径：** 累计充值满30000元，或直推10个会员 → 自动升级合伙人。

```
散客试了觉得好
    ↓
升级会员，开始推荐
    ↓
推荐的人升级了 → 你跟着升级到高级会员
    ↓
你推荐的人再推荐人 → 你的间推收入开始滚
    ↓
团队做大了 → 升级合伙人 → 被动收入覆盖你主业
```

**为什么这个飞轮能转起来？**

1. **散客→会员：** 100元体验门槛极低。用得好自然充更多。而且SEO/GEO只有7天体验，到时间了就忍不住想解锁——解锁就是升级。

2. **会员→高级会员：** 开始赚钱的人，会想赚更多。15%比10%多了50%。间推5%是一个全新的收入来源——之前没有的。

3. **高级会员→合伙人：** 团队做起来了，升级是自然的。20%+8%的佣金比例，让合伙人月收入轻松突破2万。

4. **每一层的人都在帮上一层赚钱。** 散客帮会员赚，会员帮高级会员赚，高级会员帮合伙人赚。每个人都是受益者。

**Q：软件真的永久免费？**
A：真的。我们赚的是算力消耗的钱，不是软件license的钱。你用得多我们赚得多，你不用我们一分不赚。所以软件免费，让你先用起来。

**Q：算力1元=1算力，会不会涨价？**
A：不会。消耗标准公开透明，要涨提前30天公告。而且你在系统里能看到每一算力的去向。

**Q：推荐的佣金怎么结算？**
A：月结。每月10号自动结算上月佣金。系统自动追踪，不需要你记账。

**Q：我自己不用，只推荐别人，行不行？**
A：不行。你必须自己也是用户，才有说服力。你自己都不知道产品好不好，怎么推荐给别人？散客也不能推荐——先升级会员。

**Q：合伙人要不要交加盟费？**
A：不要。升级合伙人不收钱。你的"门槛"是你自己用得多、推得多，自然达成。不卖合伙人的位置，只奖励真正在用的用户。

**Q：这是传销吗？**
A：不是。你赚的钱来自真实的算力消耗，不是拉人头。有人用就有消耗，有消耗就有分润。没人用就没钱。这不是"交钱入会"，是"用产品+推产品"。
